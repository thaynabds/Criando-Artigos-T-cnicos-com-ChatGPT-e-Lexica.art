# 🧠 Prompts Utilizados na Criação do Artigo Técnico
**Desafio de Projeto DIO — Criando Artigos Técnicos com ChatGPT e Lexica.art**  
**Autora:** Thayná Batista da Silva — Senac Recife-PE

---

## 📌 Tema Escolhido
**"IA Generativa na Prática: Como Python está mudando o futuro dos dados"**

---

## 1️⃣ PROMPT — Definir Assunto e Headline

```
Você é um escritor técnico especializado em tecnologia e inteligência artificial.
Crie 5 opções de títulos (headlines) chamativos para um artigo técnico sobre
"IA Generativa com Python para análise de dados", voltado para iniciantes e
estudantes de ADS. Os títulos devem ser curtos, instigantes e usar linguagem
acessível. Use o formato: "Como [X] está [transformando/mudando] [Y]".
```

**Headline escolhida:**  
> 🏆 "IA Generativa na Prática: Como Python está mudando o futuro dos dados"

---

## 2️⃣ PROMPT — Gerar a Estrutura do Artigo

```
Você é um especialista em IA Generativa e Python. Crie a estrutura completa de
um artigo técnico com o título "IA Generativa na Prática: Como Python está
mudando o futuro dos dados". O artigo deve ter:

- Introdução envolvente (máx. 3 parágrafos)
- 4 blocos de conteúdo técnico com subtítulos em formato de pergunta
- Exemplos de código Python em cada bloco
- 1 bloco de curiosidade ou "você sabia?"
- Conclusão com call to action para a comunidade DIO
- Tom: didático, acessível, leve e motivador
- Público: estudantes de ADS e iniciantes em programação
- Tamanho: médio (800 a 1200 palavras)
```

---

## 3️⃣ PROMPT — Escrever a Introdução

```
Escreva a introdução do artigo "IA Generativa na Prática: Como Python está
mudando o futuro dos dados". A introdução deve:
- Começar com uma pergunta provocativa ou dado impactante
- Contextualizar o momento atual da IA no mercado
- Apresentar o que o leitor vai aprender no artigo
- Ter no máximo 3 parágrafos curtos
- Tom: informal, empolgante e acolhedor para iniciantes
```

---

## 4️⃣ PROMPT — Escrever os Blocos Técnicos

```
Para o artigo sobre IA Generativa com Python, escreva o bloco técnico sobre
"[TEMA DO BLOCO]". O bloco deve:
- Ter um subtítulo em formato de pergunta
- Explicar o conceito em linguagem simples (2 parágrafos)
- Incluir um exemplo de código Python comentado e funcional
- Terminar com uma dica prática ou insight
- Máximo de 250 palavras por bloco

Blocos a criar:
1. O que é IA Generativa e por que Python virou a língua da IA?
2. Como usar a API da OpenAI com Python em 10 linhas de código?
3. Dados + IA: Como analisar planilhas com ChatGPT e pandas?
4. Prompt Engineering: A habilidade que vai diferenciar você no mercado
```

---

## 5️⃣ PROMPT — Bloco "Você Sabia?"

```
Crie um bloco curto e interessante no estilo "💡 Você Sabia?" para um artigo
sobre IA Generativa e Python. Inclua:
- 1 dado ou estatística impactante sobre o mercado de IA
- 1 curiosidade sobre Python e seu papel na IA
- Tom: leve, como uma caixa de destaque visual no artigo
- Máximo 80 palavras
```

---

## 6️⃣ PROMPT — Conclusão com Call to Action

```
Escreva a conclusão do artigo "IA Generativa na Prática: Como Python está
mudando o futuro dos dados". A conclusão deve:
- Resumir os aprendizados em 1 parágrafo
- Motivar o leitor a continuar estudando
- Incluir um call to action para: conectar no LinkedIn, acessar o GitHub
  da autora e seguir na DIO
- Mencionar o Bootcamp Bradesco GenAI & Dados da DIO
- Tom: pessoal, motivador e autêntico
- Máximo 150 palavras
```
